# protrak-client

## Tech stack

- Next.js
- React.js
- Tailwind CSS
- Apollo client

## Setup

- Installation: `npm install`
- Create a `.env` file with these values:
  ```
  # Global
  API_URL=http://localhost:4000
  GRAPHQL_SERVER_URL=http://localhost:4000/graphql
  NODE_ENV=development
  PT_ENVIRONMENT=development
  ```
